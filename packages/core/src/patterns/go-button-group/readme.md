# Button group

<!-- Description -->

Make buttons nicely grouped.

## When to use

- When you need to present users with multiple actions to choose from.
- When you need consistent alignment of buttons.

## Components

::: critical go-button dependency
Only `go-button` components are optimised as child components, use them as direct children of `go-button-group`.
:::

- [Button](../components/go-button)
