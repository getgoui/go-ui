# Nav drawer <span class="text-size-0">`go-nav-drawer`</span>

<!-- Description -->

<div class="text-size-1">Nav drawer is a component commonly used to present complex navigation structure on smaller screens.</div>

## Accessibility

This component combines the [w3c dialog](https://www.w3.org/TR/wai-aria-practices-1.1/#dialog_modal) and the [fly-out menu](https://www.w3.org/WAI/tutorials/menus/flyout/) design patterns. The keyboard focus will be trapped inside the menu when drawer is open, similarly focus will be trapped in submenu when a submenu is open.
